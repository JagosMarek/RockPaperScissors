# RockPaperScissors
Konzolová aplikace, kde hráč hraje proti počítači. Hráč si na začátku hry vybere jméno, zvolí obtížnost hry a hraje proti pc. Po ukončení hry jsou všechny zápasy vypsány. Hráč má možnost si hru zopakovat.
